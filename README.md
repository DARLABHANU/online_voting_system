# Online Voting System

This project is a full-stack web application for conducting secure, transparent, and user-friendly online elections.  
It allows users to register, vote, and view live results, while providing admin control for managing elections and candidates.

## Features

- Secure user authentication and authorization
- Multiple ongoing/past elections
- Real-time voting and result updates
- Admin dashboard for election and candidate management
- Responsive UI for desktop and mobile devices

## Technologies Used

- **Frontend:** React, TypeScript, Vite, TailwindCSS
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JWT
- **Other:** REST API design

## Getting Started

1. **Clone the repository**
    ```
    git clone https://github.com/DARLABHANU/online_voting_system.git
    ```
2. **Install dependencies**
    ```
    cd online_voting_system
    npm install
    ```
3. **Configure environment variables**  
   Set up your `.env` file with database credentials, JWT secrets, and other configs.

4. **Start the development server**
    ```
    npm run dev
    ```
